# TestAutomationAssessment

prerequisite:
1.Java 8 or above
2.Maven 3+


Clone the project from Git Hub URL 
Import the project to eclipse as maven existing project
Navigate to /TestAutomationAssessment/src/test/java/com/api/assesment/runner/CucumberRunnerTest.java
Right click and run as Junit Test
All the test will be passed for given acceptance criterias.
If need to change the expected output >>> /TestAutomationAssessment/src/test/resources/features/SampleTest.feature
